📋 Google Form Setup Required

To set up your RSVP form, follow these steps:

  1. Go to forms.google.com
  2. Create a new form with these fields:
    • Full Name (required)
    • Email Address (required)
    • Will you attend? (Yes/No/Maybe)
    • Number of Guests
    • Dietary Restrictions
  3. Go to Send (top right) → Click the <> icon to embed
  4. Copy the iframe code
  5. Replace the iframe below with your code

Google Form will appear here once you add your embed code to rsvp.html
